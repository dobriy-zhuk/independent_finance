Web Component for
Date & Time Slot Picker

A single input to pick date and time slot. Eg: Fri, 27 Nov 2020, 10 AM - 11 AM.


Features

  • Enable specific dates and time slots.
  • Allow users to select time slots like 10:30 AM - 11:00 AM.
  • Supports multi-language translations.
  • Supports multiple time slot formats. Eg: 4 PM / 4:00 PM / 16:00 / etc.
  • Useful for appointment booking, delivery time picker, restaurant reservation, etc.
  • Can also be used as a date picker without time slots.
  • Web component that's compatible across UI frameworks and Plain JS
